Você é um programador sênior, especialista em UI/UX, landing pages B2B industriais e implementação front-end moderna.

## Contexto do projeto

Este projeto é referente a uma landing page institucional para a empresa **LF PAK Equipamentos Industriais**.

A LF PAK atua no setor de **automações industriais**, com foco em soluções envolvendo:

- PLC / CLP;
- automação de máquinas industriais;
- integração de sistemas industriais;
- painéis elétricos;
- modernização de equipamentos;
- manutenção e suporte técnico;
- eficiência, produtividade e confiabilidade em processos industriais.

O objetivo é criar uma **landing page de exemplo**, visualmente profissional, moderna e convincente, utilizando como base o seguinte design system:

## Design System de referência

Utilize fielmente o design system abaixo:

`nictorys`

Referência visual / arquivo / documentação:

`@nictorys/design-system-lfpack.html`

Siga rigorosamente:

- paleta de cores;
- tipografia;
- espaçamentos;
- grid;
- botões;
- cards;
- ícones;
- bordas;
- sombras;
- estilo de imagens;
- hierarquia visual;
- componentes reutilizáveis;
- estados de hover/focus;
- responsividade.

Não misture estilos de outros design systems. A landing page deve parecer uma aplicação real construída com esse sistema visual.

## Idioma e tom de voz

Todo o conteúdo deve ser gerado em **português do Brasil**.

O tom deve ser:

- profissional;
- técnico, mas acessível;
- confiável;
- voltado para empresas industriais;
- direto e objetivo;
- com linguagem comercial B2B.

Evite frases genéricas demais. O texto deve transmitir autoridade em automação industrial.

## Objetivo da landing page

A landing page deve apresentar a LF PAK como uma empresa especializada em automações industriais, capaz de entregar soluções confiáveis para indústrias que buscam:

- reduzir paradas de produção;
- aumentar produtividade;
- modernizar máquinas;
- melhorar controle de processos;
- integrar equipamentos;
- automatizar etapas manuais;
- obter suporte técnico especializado.

A página deve incentivar o visitante a entrar em contato para solicitar orçamento, diagnóstico técnico ou conversa inicial.

## Estrutura obrigatória da landing page

Crie uma landing page completa contendo, no mínimo, as seguintes seções:

### 1. Header / Navegação

Inclua:

- logotipo textual ou espaço para logo da LF PAK;
- links de navegação:
  - Início;
  - Soluções;
  - Aplicações;
  - Sobre;
  - Contato;
- botão de CTA principal:
  - “Solicitar orçamento”
  - ou “Falar com especialista”.

O header deve ser responsivo, com menu mobile se necessário.

### 2. Hero section

Criar uma seção principal forte, com:

- headline impactante;
- subtítulo explicando o que a empresa faz;
- CTA principal;
- CTA secundário;
- imagem ou ilustração relevante.

Sugestão de direção textual:

Título:
“Automação industrial para processos mais eficientes, seguros e produtivos”

Subtítulo:
“A LF PAK desenvolve soluções em PLC, integração de sistemas e modernização de máquinas para empresas que precisam de controle, confiabilidade e performance na produção.”

CTAs:

- “Solicitar orçamento”
- “Conhecer soluções”

A imagem do hero deve representar automação industrial, painel elétrico, PLC, linhas de produção, sensores, máquinas ou integração industrial.

### 3. Seção de diferenciais

Criar cards com diferenciais da empresa, por exemplo:

- Projetos sob medida;
- Especialistas em PLC / CLP;
- Modernização de máquinas;
- Integração com sistemas industriais;
- Suporte técnico;
- Foco em produtividade e segurança.

Cada card deve ter ícone, título e descrição curta.

### 4. Seção de soluções

Criar uma seção apresentando as principais soluções da LF PAK:

- Programação de PLC / CLP;
- Desenvolvimento de painéis elétricos;
- Automação de máquinas industriais;
- Retrofit e modernização de equipamentos;
- Integração de sensores, inversores e IHMs;
- Diagnóstico e manutenção de sistemas automatizados.

Cada solução deve ser apresentada em formato visual agradável, usando cards, blocos ou layout alternado com imagem.

### 5. Seção de aplicações industriais

Mostrar onde as soluções podem ser aplicadas:

- linhas de produção;
- máquinas de embalagem;
- esteiras transportadoras;
- sistemas de envase;
- controle de motores;
- processos repetitivos;
- células industriais;
- monitoramento de equipamentos.

Essa seção deve ajudar o visitante a se identificar com o serviço.

### 6. Seção “Como trabalhamos”

Criar uma linha de processo com 4 etapas:

1. Diagnóstico técnico;
2. Planejamento da solução;
3. Implementação e testes;
4. Entrega, treinamento e suporte.

Cada etapa deve ter título, descrição e ícone.

### 7. Seção institucional / Sobre a empresa

Criar uma seção falando da LF PAK de forma profissional.

Exemplo de direção:

“A LF PAK Equipamentos Industriais atua no desenvolvimento de soluções para automação industrial, combinando conhecimento técnico, precisão na execução e foco nas necessidades reais da operação. Nosso objetivo é entregar sistemas confiáveis, seguros e preparados para aumentar a eficiência da produção.”

Não invente anos de fundação, certificações, clientes ou números específicos, a menos que sejam fornecidos.

### 8. Seção de chamada comercial

Criar uma seção de CTA mais forte antes do rodapé.

Exemplo:

“Pronto para modernizar sua operação industrial?”

Texto complementar:

“Fale com a LF PAK e descubra como a automação pode reduzir falhas, melhorar o controle do processo e aumentar a produtividade da sua empresa.”

Botão:

“Solicitar contato técnico”

### 9. Rodapé

Incluir:

- nome LF PAK Equipamentos Industriais;
- breve descrição;
- links rápidos;
- área de contato com placeholders:
  - telefone;
  - WhatsApp;
  - e-mail;
  - cidade/estado;
- direitos autorais.

Use placeholders quando dados reais não forem fornecidos.

## Imagens e elementos visuais

Gere ou referencie imagens relevantes ao setor industrial.

As imagens devem ter estilo compatível com o design system escolhido.

Direções possíveis para imagens:

- técnico analisando painel elétrico industrial;
- painel com PLC / CLP e cabos organizados;
- linha de produção automatizada;
- máquinas industriais em ambiente limpo e moderno;
- ilustração abstrata de sensores, dados e controle industrial;
- close-up de componentes elétricos industriais;
- operador acompanhando sistema supervisório ou IHM.

As imagens não devem parecer genéricas demais ou desconectadas do tema. Evite imagens com estética excessivamente corporativa genérica, como pessoas em reunião sem relação com indústria.

Caso o ambiente permita geração de imagens, gere imagens coerentes com os prompts abaixo:

### Prompt de imagem para o hero

“Ambiente industrial moderno com painel de automação aberto, PLC/CLP instalado, cabos organizados, luzes de status, máquinas industriais ao fundo, atmosfera limpa e tecnológica, iluminação profissional, estilo realista, alta qualidade, sem textos na imagem.”

### Prompt de imagem para seção de soluções

“Técnico de automação industrial analisando painel elétrico com PLC, usando notebook próximo a máquinas industriais, ambiente fabril moderno, foco em tecnologia, segurança e precisão, imagem realista, alta qualidade, sem logotipos e sem textos.”

### Prompt de imagem abstrata / complementar

“Ilustração tecnológica de automação industrial, linhas conectando sensores, PLC, motores e sistemas de controle, composição moderna, visual limpo, estilo compatível com landing page B2B industrial, sem textos.”

A página deve ser:

- responsiva;
- moderna;
- limpa;
- com boa hierarquia visual;
- otimizada para conversão;
- adequada para uma empresa técnica B2B;

- visualmente consistente com o design system informado.

## Requisitos técnicos

Implemente a landing page utilizando:

`HTML + CSS moderno (Flexbox, Grid, variáveis CSS, animações) + JavaScript para interatividade básica (menu mobile, hover effects)`

Caso a stack não seja informada, utilize uma estrutura simples e reutilizável, priorizando código limpo e fácil de adaptar.

O código deve conter:

- componentes ou seções bem organizadas;
- comentários apenas quando úteis;
- classes sem excesso desnecessário;
- boa semântica HTML;
- imagens com `alt` descritivo;
- botões e links acessíveis;
- responsividade para mobile, tablet e desktop.

## Restrições importantes

- Não usar textos em inglês.
- Não fugir do tema de automação industrial.
- Não usar imagens de tecnologia genérica sem relação com indústria.
- Não misturar estilos visuais fora do design system escolhido.
- Não criar uma página com aparência de startup SaaS genérica; ela deve parecer uma empresa técnica de equipamentos e automação industrial.
